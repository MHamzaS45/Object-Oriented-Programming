# Project Details

Your client wants an easy-to-use, installable program for their employees. The program should include at least the following:

> GUI, i.e. graphical user interface (e.g. TkInter)
> Possibility to add new tasks
> Ability to edit your own tasks
> Ability to delete your own tasks
> Logging in to the program
> To-do list according to profile
> Use a database to store tasks

<b> Plan and implement a client project creatively. Check your project into the Python repository for version control. Remember to comment your code, name variables logically, and maintain instructions in the Readme.md file. </b>
