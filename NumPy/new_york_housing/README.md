# New York Housing Data Analysis 

### Overview

A data analysis project done on analyzing housing prices in New York City, whilst signling out the cheapest ones that fall under a certain range.
Built with pandas, seaborn, numpy, matplotlib, IPython.display etc.

### How to use
1. Clone the repository 
2. Ensure file paths match the ones in the code
3. Run to achieve desired results

> To understand the workings of the code, refer to the documentation included in .ipynb file [here](https://github.com/MHamzaS45/Data-Pipelines/blob/main/NumPy/new_york_housing/nyc_housing_da.ipynb)
